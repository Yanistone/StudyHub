## Frontend

```
cd frontend
npm i
npm run dev
```

## Backend

```
cd backend
npm i
npm run dev
```

## Database

Créer la base de données:

```
npm run db:create
```

Migrer la base de données:

```
npm run db:migrate
```

Seeder la base de données:

```
npm run db:seed
```
